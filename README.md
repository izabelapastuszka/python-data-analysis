# Food Waste Analysis

Data analysis project focused on food waste patterns across countries and income groups.

## Tools used
- Python
- Pandas
- Matplotlib
- Google Colab

## Questions explored
- Do high-income countries waste more food?
- Are there regional differences in food waste?
- Is there a relationship between food waste and income group?

## Key findings
- Food waste was not directly proportional to income level.
- Significant differences were observed between regions and country groups.
